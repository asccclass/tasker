# 任務執行器


## 範例

```
package main

import(
   "time"
   "github.com/asccclass/tasker"
)

func main() {
   sch := SryTask.NewTaskScheduler()
   task := SryTask.NewTask(func() error {
      println("Hello world!")
      return nil
   })

   sch.AddTask(time.Second*1, task)
   sch.ListenForGracefullyShutdown()
   sch.Start()
}
```
